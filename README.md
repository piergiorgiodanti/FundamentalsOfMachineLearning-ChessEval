# Progetto Fundametals of Machine Learning
In questa repository è presente il codice relativo al progetto di Fundamentals of Machine Learning. 

## Addestramento
Per l'addestramento è necessario caricare nella cartella data/ il file chessData.csv.

Per la gestione delle dipendenze scaricare uv, con il seguente comando:

```bash
pip install uv
``` 

Per l'addestramento del modello, eseguire il seguente comando:

```bash
cd src/
uv run train.py
```
